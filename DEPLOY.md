# Deploying Chimitdorzhi Portfolio to GitHub Pages

Follow these steps to update your live site:

1.  **Open GitHub Desktop** or Git Bash.
2.  **Commit Changes**:
    *   Stage all files (`git add .`).
    *   Commit (`git commit -m "Add 3D background and new project"`).
3.  **Push**:
    *   Click "Push origin" (`git push`).
4.  **Verify**:
    *   Go to your repository settings -> Pages.
    *   Check the deployment status.
    *   Wait ~1-2 minutes and refresh your site.
